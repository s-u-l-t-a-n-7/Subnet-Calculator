# NetMatrix Subnetting Suite v5.1



**NetMatrix** is a professional, desktop-based IP subnetting calculator designed for network engineers, students, and IT professionals. It provides a comprehensive suite of tools for network calculation, subnet generation, and advanced VLSM planning, all wrapped in a clean and elegant user interface.

Developed by **Sultan Shamsan** with assistance from Manus AI.

---

## ✨ Features

*   **IP Calculator:** Quickly get detailed information about any IP address, including network address, broadcast address, usable host range, and more.
*   **Subnet Generator:** Generate a list of subnets from a parent network based on either the required number of hosts or the desired number of subnets.
*   **VLSM Planner:** An advanced tool to design an efficient IP addressing scheme for complex networks with multiple departments and varying host requirements.
*   **Export to CSV:** Export your generated subnet plans and VLSM layouts to a CSV file for documentation and sharing.
*   **Elegant UI:** A classic and professional user interface designed for clarity and ease of use.

## 🚀 Getting Started

You can run the application in two ways:

### 1. Download the Executable (Recommended)

The easiest way to get started is to download the pre-compiled executable file from the **[Releases Page](https://github.com/s-u-l-t-a-n-7/NetMatrix/releases )**.

1.  Go to the Releases page.
2.  Download the `NetMatrix_v5.1.exe` file.
3.  Run the file directly. No installation is needed!

### 2. Run from Source

If you have Python installed, you can run the application from the source code.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/s-u-l-t-a-n-7/NetMatrix.git
    cd NetMatrix
    ```
2.  **Run the script:**
    ```bash
    python netmatrix_app.py
    ```
